o sistema está desenvolvido com Laravel 8 e vue.js 3, para rodar o sistema tem que abrir a consola na raiz do projeto e digitar os  seguintes comandos:

1- php artisan serve (para levantar o servidor)
2- npm run watch (para disponibilizar os componentes feitos por vue.js)

para rodar esses comandos tem que ter previamente instalados node.js y composer


Funcionamento do sistema

1- O sistema contem 2 menu, um para gestão de vacinas e outro para a gestão do cadastro dos funcionarios.

2- Gestão de vacinas - tem um botão para inserir uma nova vacina e uma tabela onde se mostra todas as vacinas cadastradas no sistema com botões para editar e remover as mesmas.

3- Gestão de funcionarios - tem um botão para cadastrar um novo funcionario e uma tabela onde se mostra todas os funcionarios cadastradas no sistema com botões para editar e remover as mesmas.
